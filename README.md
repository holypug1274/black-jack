# REQUIREMENTS

-   Node Js (https://nodejs.org/en/) - v14.18.1
-   Git (https://git-scm.com/downloads) - 2.33.1

# PREPARE THE PROJECT

```bash
git clone https://github.com/holypug1274/black-jack.git
```

# RUN THE PROJECT

```bash
npm ci
npm run build
npm start
```

# TESTING

1. Install dependencies

```bash
npm test
```
